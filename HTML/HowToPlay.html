Controls

Arcade Mode

Arcade Mode (AKA Time Attack) allows the user to set a number of levels they wish to complete. The player's goal is to finish all the levels in the best time they can.


Number of Levels: Put a positive number in this field to specify how many levels you want to play in a row.


Max Width: Put a positive number in this field to specify the largest width the picross board can be. Boards will be randomized between the minimum width and this value for the width.


Max Height: Put a positive number in this field to specify the largest height the picross board can be. Boards will be randomized between the minimum height and this value for the height.


Minimum Width: Put a positive number in this field to specify the smallest width the picross board can be. Boards will be randomized between this value and the maximum width value for the final width.


Minimum Height: Put a positive number in this field to specify the smallest height the picross board can be. Boards will be randomized between this value and the maximum height value for the final height.
