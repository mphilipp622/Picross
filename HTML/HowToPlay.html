What is Picross?

I recommend referring to the Wikipedia Article about nonograms to better understand the basics of Picross.


Arcade Mode

Arcade Mode (AKA Time Attack) allows the user to set a number of levels they wish to complete. The player's goal is to finish all the levels in the best time they can.

The user has the option to choose the number of levels they wish to play in a row. You can also choose the grid size you wish to play on.

Custom Mode

Custom mode allows the user to play a single grid of 7x7 or 13x13 size.

When the player chooses this mode, a grid is randomly generated. Additionally, the player can click the "next level" and "previous level" buttons on the "Save/Load Levels" bar on the right side of the page to select a level to play.

Image Mode

Image mode allows you to upload an image and have it converted into a 7x7 or 13x13 picross grid. Simply choose a file using the "Choose Image" button and then click "Create grid from image" and you're good to go.
